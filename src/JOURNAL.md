- Программа была разделена на модули [db_module.py](db_module.py) для инициализации БД и таблиц, [analyzer_module.py](analyzer_module.py) для хранения функций для анализа, т.е. find_duplicates() и check_history_report(), [scanner_module.py](scanner_module.py) с основной логикой работы программы и входная точка [main.py](main.py)

## 26.06.2026
- Добавлена возможность указывать аргументом папку бэкапа
- Добавлена функция для сравнения основной папки и бэкапа а тажке вывода различий если они есть
## 30.06.2026
- Добавлена таблица БД **backup_history** с историей сравнений с бэкап папкой в модуле [db_module.py](db_module.py)
- В таблице **files** в качестве первичного ключа взят составной из пути к папке и пути к файлу внутри папки.
- В метод *compare_directories* из [analyzer_module.py](analyzer_module.py) добавлена логика добавления записей в таблицу **backup_history**
- В модуль [main.py](main.py) введены следующие изменения:
  1. Возможность работы без указания папки бэкапа, то есть просто проиндексировать рабочую папку без сравнения
  2. Добавлен вывод ошибок при различный сценариях с количеством аргументов и успеха в поиске папок
